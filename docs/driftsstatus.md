# Driftsstatus

## Servicevinduer og kritiske fejl
**OBS! Udenfor valgperioder opdateres denne side ikke!**
Hvis du oplever problemer med OS2valghalla, så tjek i første omgang [oversigten over kendte fejl](https://os2web.atlassian.net/wiki/spaces/OS2valghalla/pages/2379612161/Kendte+fejl+i+OS2valghalla).

Hvis du oplever andre fejl, så indrapporter det på [denne formular](https://www.os2.eu/os2valghalla-fejlrapportering).

Kritiske fejl indrapporteres til <os2valghalla@os2.eu>.

Kommunens ansvarlige for OS2valghalla har også mulighed for at kontakte Precio Fishbone AB direkte med driftsfejl.

Udviklingsønsker kan indrapporteres på [denne formular](https://www.os2.eu/os2valghalla-udviklingsonsker).


## Hjælp til fejlsøgning

Samlet aktuel driftsstatus på [Digitaliser](https://www.digitaliser.dk/).

### Login

- Hvis der er problemer med MitID, kan det skyldes problemer hos:
  - NemLog-in - [driftsstatus](https://www.digitaliser.dk/nemlog-in)
  - MitID - [driftsstatus](https://www.digitaliser.dk/mitid/driftsstatus-for-mitid)
- Hvis der er problemer med at logge ind i den administrative hjemmeside, kan det skyldes problemer hos KOMBIT. Se efter oplysninger om "Fælleskommunal Adgangsstyring for brugere (Context Handler 2)" ved [KOMBIT driftsstatus](https://driftstatus.kombit.dk/)


### Digital Post

- Hvis der er problemer med Digital Post, kan det skyldes problemer hos:
  - Digital Post - [driftsstatus](https://www.digitaliser.dk/digital-post/driftsstatus)
  - KOMBIT - se efter oplysninger om "SF1601 Afsend post", "SF1461 Modtag beskeder via Beskedfordeler" og "Fælleskommunal Beskedfordeler" ved [KOMBIT driftsstatus](https://driftstatus.kombit.dk/)


### CPR-opslag
- Hvis der er problemer med CPR-opslag, kan det skyldes problemer hos KOMBIT. Se efter oplysninger om "SF1520 - CPR replika opslag" ved [KOMBIT driftsstatus](https://driftstatus.kombit.dk/)
