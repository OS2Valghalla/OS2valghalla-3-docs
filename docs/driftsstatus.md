# Driftsstatus
Servicevinduer og kritiske fejl
OBS! Udenfor valgperioder opdateres denne side ikke!

 

Hvis du oplever problemer med OS2valghalla, så tjek i første omgang oversigten over kendte fejl.

Hvis du oplever andre fejl, så indrapporter det på denne formular.

Kritiske fejl indrapporteres til os2valghalla@os2.eu.

Kommunens ansvarlige for OS2valghalla har også mulighed for at kontakte Omnia/Precio Fishbone direkte med driftsfejl.

Udviklingsønsker kan indrapporteres på denne formular.
