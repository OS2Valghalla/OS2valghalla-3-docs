title: "Registrering af OS2valghalla som NemLog-in system"
parent: "Vejledninger"
has_children: false
---

Kommunen skal registrere OS2valghalla i portalen NemLog-In Administration, så deltagere kan logge ind i den eksterne hjemmeside med MitID.

Kommunen skal i denne forbindelse forbindelse have en NSIS-vurdering af sikkerhedsniveau. OS2 har i
samarbejde med Københavns Kommune udarbejdet dette skema med vurderingen, som I bør downloade og
gemme.

Bemærk at denne registrering er nødvendig, selvom Fælleskommunal Adgangsstyring/KOMBIT Context
Handler 2 er opsat.
